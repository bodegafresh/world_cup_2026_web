# Pool Team 2026 🎱⚽

Dashboard privado con predicciones y estadísticas del Mundial FIFA 2026 para el grupo Pool Team.

## Setup

1. Edita `js/config.js` y pon tu Cloudflare Worker URL
2. Sube el repo a GitHub
3. Activa GitHub Pages: Settings → Pages → Branch: main → Folder: / (root)

## Acceso

La página pide clave de acceso. La clave es la misma que configuraste como `WEB_KEY` en Cloudflare y `WEB_SECRET_KEY` en Google Apps Script.
